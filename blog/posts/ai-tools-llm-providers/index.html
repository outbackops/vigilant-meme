    Cloud Provider Offerings¶

    Azure OpenAI¶

    Enterprise-grade OpenAI with Azure integration.

    Feature Details
    Models Same as OpenAI (GPT-4o, etc.)
    Compliance SOC 2, HIPAA, GDPR, etc.
    SLA 99.9% uptime guarantee
    Data Your data stays yours, not used for training
    Integration Native Azure ecosystem

    When to choose: - Enterprise compliance requirements - Existing Azure infrastructure - Need SLAs and support contracts - Data residency requirements

    # Quick start
from openai import AzureOpenAI
client = AzureOpenAI(
    azure_endpoint="https://YOUR-RESOURCE.openai.azure.com",
    api_key="YOUR_KEY",
    api_version="2024-02-15-preview"
)
